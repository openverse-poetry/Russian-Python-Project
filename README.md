# Русский Питон v1.0.0

Промышленный компилятор языка программирования с русским синтаксисом.

## Возможности

- Полный русский синтаксис для всех конструкций Python
- Совместимость с экосистемой Python
- Лексер, парсер, транслятор в Python AST
- Поддержка функций, классов, условий, циклов, исключений

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
# Токенизация
python src/core/lexer.py

# Парсинг
python src/core/parser.py

# Трансляция
python src/backend/translator.py
```

## Пример

```русский
функция привет(имя):
    печать(f"Привет, {имя}!")
    если имя == "Мир":
        возврат истина
    иначе:
        возврат ложь
```
